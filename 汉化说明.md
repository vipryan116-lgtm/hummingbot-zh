# Hummingbot 汉化说明

本仓库已支持**简体中文**界面，通过环境变量切换语言。

## 启用中文

在**启动 Hummingbot 之前**设置环境变量：

**Windows (PowerShell):**
```powershell
$env:HUMmingBOT_LANG = "zh_CN"
python bin/hummingbot_quickstart.py
```

**Windows (CMD):**
```cmd
set HUMmingBOT_LANG=zh_CN
python bin\hummingbot_quickstart.py
```

**Linux / macOS:**
```bash
export HUMmingBOT_LANG=zh_CN
./bin/hummingbot_quickstart.py
```

也可使用 `zh` 或 `zh-CN`，效果相同。

## 恢复英文

不设置 `HUMmingBOT_LANG` 或设为 `en` 即为英文：

```powershell
$env:HUMmingBOT_LANG = "en"
```

## 已汉化内容

- 启动界面欢迎语与常用命令说明
- 顶部状态栏：版本、策略、策略文件、网关
- 运行日志、搜索提示
- 所有主命令与子命令的帮助说明（connect、create、import、help、balance、config、start、stop、status、history、gateway、exit、export、ticker、mqtt、rate 等）

## 技术说明

- 翻译模块：`hummingbot/client/i18n.py`
- 英文与简体中文文案均在上述文件中维护，可自行增改
- 当前语言由环境变量 `HUMmingBOT_LANG` 决定，进程内可通过 `hummingbot.client.i18n.set_language("zh_CN")` 修改（仅影响当前进程）

## Docker 使用汉化版

**不需要先上传 GitHub**，在本地用汉化后的代码构建镜像即可。

### 方式一：本地构建镜像（推荐，不依赖 GitHub）

1. 在项目根目录（汉化后的 hummingbot 目录）执行构建：
   ```bash
   docker build -t hummingbot-zh .
   ```
2. 修改 `docker-compose.yml`：把 `image: hummingbot/hummingbot:latest` 改为使用本地镜像，并传入中文环境变量。例如：
   ```yaml
   services:
     hummingbot:
       container_name: hummingbot
       image: hummingbot-zh   # 使用上面 build 的镜像
       environment:
         - HUMmingBOT_LANG=zh_CN
       volumes:
         # ... 保持原有 volumes
   ```
3. 启动并进入容器：
   ```bash
   docker compose up -d
   docker attach hummingbot
   ```

### 方式二：先上传到 GitHub，再在服务器/CI 上构建

若你希望在另一台机器或通过 CI 构建，可先把汉化代码推到你的 GitHub 仓库，然后在那台机器上：

```bash
git clone https://github.com/你的用户名/你的仓库名.git
cd 你的仓库名
docker build -t hummingbot-zh .
# 同上，在 docker-compose 里用 hummingbot-zh 并设置 HUMmingBOT_LANG=zh_CN
```

## 上传到你的 GitHub

汉化已提交到本地（commit: feat: add Simplified Chinese UI）。若要推送到你的 GitHub 仓库：

1. 在 GitHub 网页上**新建一个仓库**（如 `hummingbot-zh`），不要勾选 “Add a README”。
2. 在本地项目目录执行（把 `YOUR_USERNAME`、`YOUR_REPO` 换成你的**用户名**和**仓库名**）：
   ```bash
   git remote add mygithub https://github.com/YOUR_USERNAME/YOUR_REPO.git
   git push -u mygithub master
   ```
   若 GitHub 上该仓库默认分支是 `main`，则改为：`git push -u mygithub master:main`
3. 之后更新汉化时：改完代码后 `git add` → `git commit` → `git push mygithub master` 即可。
